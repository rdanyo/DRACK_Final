The ProtoSite Rationale:

Based on the research of the students questionnaire and other discussions our team agreed that the current COMP1950 website was informative, easy to navigate and content was displayed in logical order but at the same time certain improvements could be made such as more useful navigation for easier access to resources, a fresher look, and a better mobile experience.

The Target Audience:

  • Current COMP1950 students
  • Prospective students for section of the page
  • Alumni students (maybe provide an area for access once course completed)

Characteristics of the Current Site:

  • Simplistic design with no clutter.
  • Content in logical order.
  • Easy to read and navigate.
  • No media query for mobile.
  • Reference material, exercises and assignments are within content only.
  • Outdated look.

Proposed ProtoSite:

  • Add Search form on top of site for easy access to specific topics.
  • Main Navigation with drop-down menu moved to the top of page allowing user-friendly access.
  • Leaving various components such as resources, references, exercises, assignments, code samples within the content but at the same time providing fast access by integrating them into the main navigation section.
  • Keeping the lecture notes.
  • Keeping same or similar information architecture as content is already well laid out.
  • Adding a blog page with up-to-date news, shared articles, posts.
  • @media query to allow for mobile viewing.
  • Update the look and feel with fresher, more modern design, different color features as well as imagery without loosing focus on content and readability.
  • Bigger font size and background color for improved readability.
  • Create access as resource tool for Alumni students.
  • For faster and more accessible content update the ProtoSite could easily be transitioned into a CMS but at this point we are not concerned as the client is a highly experienced web developer.
